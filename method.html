  1. Simulation Condition

    We set the binding rate constant (kf) based on toehold length as follows.

    If toehold_length ≤ 6
       kf = 5.0×10^(n-1) [M^(-1)s^(-1)]
    else
       kf = 3.0×10^6 [M^(-1)s^(-1)]
    Where
       n : toehold_length

  3. Wet Experiment of DNA Tractor

    希釈

    アニーリング

    電気泳動

    ① 電気泳動装置（AE-y530MWラピダス・ミニスラブ電気泳動装置（アトー））にゲル（e-PAGEL）をセットした。
    ② サンプル（10μL）とローディングダイ（Gel Loading Dye Purple (6x) B7024S (BioLabs) 4mL）（2μL）を混合した。
    ③ ゲルのレーンに②で作成したサンプルをそれぞれ（10μL）注入し、電気泳動を（80分）行った。
    ④ ゲルを取り出し、DNA染色試薬（ GelGreen Nucleic Acid Stain 10,000X DMSO 0.5mL）で40分染色した。
    ⑤ 染色したゲルをゲルイメージャーで観察、分析した。
