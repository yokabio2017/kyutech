YOKABIO PROJECT

Background

近年、DNAナノテクノロジーの発展によりDNAオリガミやDNAタイルを組み合わせて様々な構造物が作られている。さらに、DNAナノコンピューティングの分野と融合しDDSに代表されるような分子デバイスの開発が行われている。例えば、２次元DNAオリガミを組み合わせることで箱型のDNAオリガミ構造体が作られ、箱の上部はDNAの鎖置換反応によって開け閉めが可能となり蓋としての機能を果たしている[1]。しかし、この蓋の制御法では完全に閉じるか、完全に開くかの２状態しか存在せず、半分だけ開けるといった開き具合の調整ができない。このような問題点を基に、我々はDNAオリガミ構造体を任意の距離だけ動かす方法が構築できれば、蓋の開け閉めのみならず、さまざまなアプリケーションに応用が出来るのではないかと考えた。

Recently, DNA nanotechnology has evolved to combine DNA origami and DNA tiles to create various structures. In addition, molecular devices typified by DDS have been developed that combine with the field of DNA nanocomputing. For example, by combining two-dimensional DNA origami, a box-shaped DNA origami structure is made, and the upper part of the box is capable of opening and closing by DNA strand substitution reaction and fulfills the function as a lid[1]. However, according to the control method of this lid, there are only two states of completely closing or fully opening, and it is impossible to adjust the degree of opening such that only half can be opened. Based on these problems, we thought that if we could build a method to move the DNA origami structure by an arbitrary distance, we could apply it not only to the opening and closing of the lid but also to various applications.

Project Idea

我々は、DNA構造物を動かすための動力源となるシステムを作り上げることにした。今回、動かす対象となるDNA構造物とその動力源が一体となったシステムではなく、動かす対象となる構造物とその動力源を切り離して考えた。その結果、アプリケーションの仕様によってDNA構造物の構造が変わったとしても動力源となるシステムと接続さえできれば動かすことが出来ると考えた。したがって、本プロジェクトでは動力源となるシステムの設計を目標とした。今回提案する動力源となるシステムは以下の4つの仕様を満たす必要がある。 1つ目は対象となるDNA構造物と接続可能であること。 2つ目は自律的に駆動するシステムであること。 3つ目は溶液中でDNA分子はブラウン運動をしており四方八方に行き来してしまうため、1方向にのみ進むようにすること。 そして4つ目は進み具合を調整できること。 我々は、DNAWalkerと呼ばれる構造物[2]の動作原理を参考にして、これら４つの仕様を満たすようなシステムを構築した。

We decided to create a system that is a power source for moving DNA structures. In this time, we thought about separating the structure to be moved and its power source, not the system in which the DNA structure to be moved and its power source are united. As a result, we thought that even if the structure of the DNA structure was changed according to the specification of the application, the DNA structure could move as long as it could connect with the system as the power source. Therefore, this project aimed at designing a system to be a power source. The system to be proposed as the power source must satisfy the following four specifications.
First, this system can connect with the target DNA structure.
Second, this system can drive autonomously.
Third, since DNA molecules are in Brownian motion in solution, they move in all directions, so this system should go only in one direction.
Fourth, this system can adjust their progress.
We built a system that satisfies these four specifications with reference to the principle of operation of a structure called DNAWalker[2].
